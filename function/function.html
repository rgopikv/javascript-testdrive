Javascript code:

	function sayHelloWithName( myName )
	{
		if( "" != myName )
			alert( 'Hello ' + myName + '!' );
		else
			alert( 'Please enter your name!' );
	}

'nested' function in JavaScript. Function with in a function is called nested function.

Javascript code:

	function nestedFunction( a, b )
	{
		function square( x )
		{
			return x * x;
		}

		return Math.sqrt( square( a ) + square( b ) );
	}

Function literal

Javascript code:

	var func = function( a, b ){ return a + b };
	
	function callFunctoinLiteral( a, b )
	{
		return func( parseInt( a ), parseInt( b ) );
	}
